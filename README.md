# Baby Names Suggestion Tool

A simple tool to collect baby name suggestions from family and friends, automatically organizing them by **Gujarati zodiac (Rashi) starting letters** while avoiding duplicates and keeping the process collaborative.

---

## Background

In Gujarati culture, a baby's name traditionally starts with a letter determined by the **Rashi (zodiac sign)** based on the time and date of birth.

When asking family and friends for suggestions, the process usually ends up scattered across:

- WhatsApp messages
- Notes
- Documents
- Phone conversations

This makes it difficult to:

- Track all suggestions
- Avoid duplicate names
- Filter names based on the correct zodiac letters

This project was created to **simplify and organize that process**.

---

## What This Tool Does

This tool helps you:

- Collect baby name suggestions from family and friends
- Automatically organize names based on **Gujarati Rashi letters**
- Avoid duplicate name suggestions
- Keep everything in **one clean and structured place**

---

## Why I Built This

Instead of maintaining a single document and manually:

- figuring out the correct zodiac letters
- removing duplicates
- organizing suggestions

I created this small tool so that **everyone can submit their suggestions easily**, while the system keeps everything sorted.

---

## How It Can Be Used

1. Share the tool with family and friends.
2. Let them submit their baby name suggestions.
3. Names get organized automatically based on **Gujarati zodiac letters**.
4. Review and shortlist names easily.
